# 🛠️ Melhoria Contínua Wiki

A dynamic, JSON-driven knowledge hub designed to bridge the gap between traditional Operational Excellence frameworks and modern Local AI automation.

> **🇧🇷 Nota:** Embora este repositório utilize documentação em Inglês para fins de portfólio técnico, o conteúdo da aplicação e os estudos de caso foram desenvolvidos em **Português** para atender ao contexto corporativo brasileiro.

---

## 🚀 The Vision
This project is not just a website; it's a **Living Framework**. It was built to organize complex methodologies like Lean, Six Sigma, and BPMN into a pedagogical and actionable interface. 

The goal is to provide a "Single Source of Truth" for process optimization, featuring practical case studies that integrate **Python automation** and **Local AI (Ollama/Gemma)**.

## 🏗️ Architecture: Data-Driven & Local-First
To ensure scalability and clean separation of concerns, the project follows a **JSON-driven architecture**:
- **Core Engine:** Vanilla JavaScript/CSS3 for zero-dependency, high-speed rendering.
- **Data Layer:** A centralized `data.js` (JSON) file that allows adding new methodologies and case studies without touching the UI logic.
- **Deployment:** Continuous Deployment via Netlify.

## 📚 Methodologies Included
- **SIPOC / VSM:** For macro and micro process mapping.
- **Ishikawa (6M) & 5 Whys:** For root cause analysis.
- **Poka-Yoke:** Focused on systemic error-proofing via code.
- **Six Sigma (Statistical Control):** Applied to liquidity and risk management.
- **Local AI Integration:** Practical use of LLMs (Gemma/Llama) for data classification and compliance.
